# Индикаторы набора текста

Индикаторы набора текста отправляются в чат во время активного выполнения задачи. Используйте `agents.defaults.typingMode`, чтобы управлять **тем, когда** начинается отображение индикатора набора, и `typingIntervalSeconds`, чтобы управлять **тем, как часто** он обновляется.

## Значения по умолчанию

Если параметр `agents.defaults.typingMode` **не задан**, OpenClaw использует устаревшее поведение:

- **Личные чаты**: индикатор набора появляется сразу, как только начинается цикл работы модели.
- **Групповые чаты с упоминанием**: индикатор набора появляется сразу.
- **Групповые чаты без упоминания**: индикатор набора появляется только тогда, когда начинается потоковая передача текста сообщения.
- **Выполнение heartbeat**: индикатор набора отключён.

## Режимы

Установите для `agents.defaults.typingMode` одно из следующих значений:

- `never` — индикатор набора никогда не отображается.
- `instant` — индикатор набора появляется **сразу, как только начинается цикл работы модели**, даже если впоследствии задача возвращает только токен молчаливого ответа.
- `thinking` — индикатор набора появляется при **первом дельта-изменении рассуждений** (требуется `reasoningLevel: "stream"` для выполнения задачи).
- `message` — индикатор набора появляется при **первом немолчаливом дельта-изменении текста** (игнорирует токен молчаливого ответа `NO_REPLY`).

Порядок "насколько рано срабатывает":
`never` → `message` → `thinking` → `instant`

## Настройка

```json5
{
  agent: {
    typingMode: "thinking",
    typingIntervalSeconds: 6,
  },
}
```

Вы можете переопределить режим или частоту обновления для каждой сессии:

```json5
{
  session: {
    typingMode: "message",
    typingIntervalSeconds: 4,
  },
}
```

## Примечания

- В режиме `message` индикатор набора не отображается для ответов, состоящих только из молчаливого токена (например, `NO_REPLY` / `no_reply`, сопоставление без учёта регистра).
- Режим `thinking` срабатывает только в том случае, если выполнение задачи включает потоковую передачу рассуждений (`reasoningLevel: "stream"`). Если модель не выдаёт дельта-изменения рассуждений, индикатор набора не появится.
- Для heartbeat индикатор набора никогда не отображается, независимо от режима.
- Параметр `typingIntervalSeconds` управляет **частотой обновления**, а не временем начала отображения. Значение по умолчанию — 6 секунд.
