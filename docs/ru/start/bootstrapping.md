# Начальная загрузка агента

Начальная загрузка — это процедура **первого запуска**, которая подготавливает рабочее пространство агента и собирает данные об идентификации. Она выполняется после подключения, когда агент запускается в первый раз.

## Что делает начальная загрузка

При первом запуске агента OpenClaw выполняет начальную загрузку рабочего пространства (по умолчанию `~/.openclaw/workspace`):

- Заполняет файлы `AGENTS.md`, `BOOTSTRAP.md`, `IDENTITY.md`, `USER.md`.
- Проводит краткий ритуал вопросов и ответов (по одному вопросу за раз).
- Записывает данные об идентификации и предпочтениях в файлы `IDENTITY.md`, `USER.md`, `SOUL.md`.
- Удаляет файл `BOOTSTRAP.md` по завершении, чтобы процедура выполнялась только один раз.

## Где выполняется начальная загрузка

Начальная загрузка всегда выполняется на **хосте шлюза**. Если приложение для macOS подключается к удалённому шлюзу, рабочее пространство и файлы начальной загрузки находятся на этом удалённом компьютере.

<Note>
Когда шлюз работает на другом компьютере, редактируйте файлы рабочего пространства на хосте шлюза (например, `user@gateway-host:~/.openclaw/workspace`).
</Note>
