# BOOT.md

Добавьте краткие чёткие инструкции о том, что должен делать OpenClaw при запуске (включите `hooks.internal.enabled`).
Если задача отправляет сообщение, используйте инструмент отправки сообщений, а затем ответьте точным тихим токеном `NO_REPLY` / `no_reply`.
