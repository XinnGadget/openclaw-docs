# AGENTS.md — рабочее пространство OpenClaw

Эта папка является рабочим каталогом ассистента.

## Первый запуск (однократно)

- Если существует файл BOOTSTRAP.md, выполните указанные в нём действия и удалите файл после завершения.
- Идентификатор вашего агента хранится в файле IDENTITY.md.
- Ваш профиль хранится в файле USER.md.

## Совет по резервному копированию (рекомендуется)

Если вы рассматриваете это рабочее пространство как "память" агента, сделайте из него репозиторий Git (в идеале — приватный), чтобы сохранить идентификатор и заметки.

```bash
git init
git add AGENTS.md
git commit -m "Добавить рабочее пространство агента"
```

## Настройки безопасности по умолчанию

- Не выгружайте секреты или личные данные.
- Не выполняйте деструктивные команды, если это явно не запрошено.
- Будьте лаконичны в чате; более развёрнутые выводы записывайте в файлы в этом рабочем пространстве.

## Ежедневная память (рекомендуется)

- Ведите краткий ежедневный журнал в файле memory/YYYY-MM-DD.md (при необходимости создайте папку memory/).
- В начале сессии читайте записи за сегодняшний и вчерашний день (если они есть).
- Фиксируйте устойчивые факты, предпочтения и решения; избегайте секретов.

## Пульсации (необязательно)

- В файле HEARTBEAT.md можно разместить краткий чек-лист для запусков в режиме пульсации; держите его небольшим.

## Настройка

- Добавьте сюда свой предпочтительный стиль, правила и "память".
