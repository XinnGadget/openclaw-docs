# fal

В OpenClaw включён провайдер `fal` для удалённой генерации изображений и видео.

- Провайдер: `fal`
- Аутентификация: `FAL_KEY` (основной вариант; в качестве запасного также работает `FAL_API_KEY`)
- API: конечные точки моделей fal

## Быстрый старт

1. Установите ключ API:

```bash
openclaw onboard --auth-choice fal-api-key
```

2. Задайте модель изображения по умолчанию:

```json5
{
  agents: {
    defaults: {
      imageGenerationModel: {
        primary: "fal/fal-ai/flux/dev",
      },
    },
  },
}
```

## Генерация изображений

Провайдер генерации изображений `fal` по умолчанию использует модель `fal/fal-ai/flux/dev`.

- Генерация: до 4 изображений за запрос
- Режим редактирования: включён, 1 эталонное изображение
- Поддерживаются параметры `size`, `aspectRatio` и `resolution`
- Текущее ограничение режима редактирования: конечная точка редактирования изображений fal **не поддерживает** переопределение `aspectRatio`

Чтобы использовать fal в качестве провайдера изображений по умолчанию:

```json5
{
  agents: {
    defaults: {
      imageGenerationModel: {
        primary: "fal/fal-ai/flux/dev",
      },
    },
  },
}
```

## Генерация видео

Провайдер генерации видео `fal` по умолчанию использует модель `fal/fal-ai/minimax/video-01-live`.

- Режимы: text-to-video и поток с использованием одного эталонного изображения
- Время выполнения: поток отправки/проверки статуса/получения результата с использованием очереди для длительных задач

Чтобы использовать fal в качестве провайдера видео по умолчанию:

```json5
{
  agents: {
    defaults: {
      videoGenerationModel: {
        primary: "fal/fal-ai/minimax/video-01-live",
      },
    },
  },
}
```
