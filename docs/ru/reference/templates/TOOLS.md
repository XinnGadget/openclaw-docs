# TOOLS.md — Локальные заметки

Навыки определяют, _как_ работают инструменты. Этот файл предназначен для _ваших_ специфических данных — того, что уникально для вашей настройки.

## Что размещать здесь

Например:

- Названия и расположение камер
- Хосты и псевдонимы SSH
- Предпочитаемые голоса для TTS
- Названия колонок/помещений
- Прозвища устройств
- Любые данные, специфичные для окружения

## Примеры

```markdown
### Камеры

- living-room → Основная зона, широкоугольный объектив 180°
- front-door → Вход, с активацией по движению

### SSH

- home-server → 192.168.1.100, пользователь: admin

### TTS

- Предпочитаемый голос: "Nova" (тёплый, с лёгким британским акцентом)
- Колонка по умолчанию: Kitchen HomePod
```

## Зачем разделять?

Навыки являются общими. Ваша настройка — ваша собственная. Разделение позволяет обновлять навыки без потери ваших заметок и делиться навыками, не раскрывая свою инфраструктуру.
