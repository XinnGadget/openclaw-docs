# Поиск через DuckDuckGo

OpenClaw поддерживает DuckDuckGo в качестве провайдера `web_search` **без ключа**. API-ключ или учётная запись не требуются.

<Warning>
  DuckDuckGo — это **экспериментальная, неофициальная** интеграция, которая получает результаты с не-JavaScript-страниц поиска DuckDuckGo, а не через официальный API. Возможны периодические сбои из-за страниц с проверками для ботов или изменений в HTML.
</Warning>

## Настройка

API-ключ не нужен — просто установите DuckDuckGo в качестве провайдера:

<Steps>
  <Step title="Настроить">
    ```bash
    openclaw configure --section web
    # Выберите "duckduckgo" в качестве провайдера
    ```
  </Step>
</Steps>

## Конфигурация

```json5
{
  tools: {
    web: {
      search: {
        provider: "duckduckgo",
      },
    },
  },
}
```

Дополнительные настройки уровня плагина для региона и SafeSearch:

```json5
{
  plugins: {
    entries: {
      duckduckgo: {
        config: {
          webSearch: {
            region: "us-en", // код региона DuckDuckGo
            safeSearch: "moderate", // "strict", "moderate" или "off"
          },
        },
      },
    },
  },
}
```

## Параметры инструмента

| Параметр | Описание |
| ------------ | ---------------------------------------------------------- |
| `query` | Поисковый запрос (обязательный) |
| `count` | Количество возвращаемых результатов (1–10, по умолчанию: 5) |
| `region` | Код региона DuckDuckGo (например, `us-en`, `uk-en`, `de-de`) |
| `safeSearch` | Уровень SafeSearch: `strict`, `moderate` (по умолчанию) или `off` |

Регион и SafeSearch также можно задать в конфигурации плагина (см. выше) — параметры инструмента переопределяют значения конфигурации для каждого запроса.

## Примечания

- **Без API-ключа** — работает сразу, без настройки
- **Экспериментальный** — получает результаты с не-JavaScript-HTML-страниц поиска DuckDuckGo, а не через официальный API или SDK
- **Риск проверок для ботов** — DuckDuckGo может выдавать CAPTCHA или блокировать запросы при интенсивном или автоматизированном использовании
- **Парсинг HTML** — результаты зависят от структуры страницы, которая может измениться без предупреждения
- **Порядок автоматического определения** — DuckDuckGo является первым провайдером по умолчанию без ключа (порядок 100) при автоматическом определении. Провайдеры на основе API с настроенными ключами запускаются первыми, затем — Ollama Web Search (порядок 110), затем — SearXNG (порядок 200)
- **SafeSearch по умолчанию имеет уровень moderate**, если не настроен

<Tip>
  Для продакшена рассмотрите [Brave Search](/tools/brave-search) (доступен бесплатный тариф) или другого провайдера на основе API.
</Tip>
