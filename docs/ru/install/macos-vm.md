# OpenClaw на виртуальных машинах macOS (песочница)

## Рекомендуемый вариант по умолчанию (для большинства пользователей)

- **Небольшой Linux VPS** для постоянно работающего шлюза с низкими затратами. См. [хостинг VPS](/vps).
- **Выделенное оборудование** (Mac mini или Linux-система), если вам нужен полный контроль и **резидентский IP-адрес** для автоматизации браузера. Многие сайты блокируют IP-адреса дата-центров, поэтому локальный просмотр часто работает лучше.
- **Гибридный вариант:** оставьте шлюз на недорогом VPS и подключайте Mac в качестве **узла**, когда вам нужна автоматизация браузера/пользовательского интерфейса. См. [Узлы](/nodes) и [Удалённый шлюз](/gateway/remote).

Используйте виртуальную машину macOS, если вам конкретно нужны возможности, доступные только в macOS (iMessage/BlueBubbles), или вы хотите строгой изоляции от вашего повседневного Mac.

## Варианты виртуальных машин macOS

### Локальная ВМ на вашем Mac с Apple Silicon (Lume)

Запустите OpenClaw в изолированной виртуальной машине macOS на вашем существующем Mac с Apple Silicon с помощью [Lume](https://cua.ai/docs/lume).

Это даёт вам:
- полную среду macOS в изоляции (ваш хост остаётся чистым);
- поддержку iMessage через BlueBubbles (невозможно на Linux/Windows);
- мгновенный сброс путём клонирования ВМ;
- отсутствие дополнительных затрат на оборудование или облако.

### Поставщики размещённых Mac (облако)

Если вам нужна macOS в облаке, подойдут поставщики размещённых Mac:
- [MacStadium](https://www.macstadium.com/) (размещённые Mac);
- другие поставщики размещённых Mac также подходят — следуйте их документации по ВМ и SSH.

Как только у вас появится доступ по SSH к виртуальной машине macOS, переходите к шагу 6 ниже.

---

## Быстрый путь (Lume, для опытных пользователей)

1. Установите Lume.
2. `lume create openclaw --os macos --ipsw latest`.
3. Завершите работу мастера настройки, включите удалённый вход (SSH).
4. `lume run openclaw --no-display`.
5. Подключитесь по SSH, установите OpenClaw, настройте каналы.
6. Готово.

---

## Что вам нужно (Lume)

- Mac с Apple Silicon (M1/M2/M3/M4);
- macOS Sequoia или новее на хосте;
- около 60 ГБ свободного места на диске на каждую ВМ;
- около 20 минут.

---

## 1) Установите Lume

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/trycua/cua/main/libs/lume/scripts/install.sh)"
```

Если `~/.local/bin` нет в вашем PATH:

```bash
echo 'export PATH="$PATH:$HOME/.local/bin"' >> ~/.zshrc && source ~/.zshrc
```

Проверьте:

```bash
lume --version
```

Документация: [Установка Lume](https://cua.ai/docs/lume/guide/getting-started/installation)

---

## 2) Создайте виртуальную машину macOS

```bash
lume create openclaw --os macos --ipsw latest
```

Это загрузит macOS и создаст ВМ. Окно VNC откроется автоматически.

Примечание: загрузка может занять некоторое время в зависимости от вашего соединения.

---

## 3) Завершите работу мастера настройки

В окне VNC:
1. Выберите язык и регион.
2. Пропустите ввод Apple ID (или войдите, если хотите использовать iMessage позже).
3. Создайте учётную запись пользователя (запомните имя пользователя и пароль).
4. Пропустите все дополнительные функции.

После завершения настройки включите SSH:
1. Откройте "Системные настройки" → "Основные" → "Общий доступ".
2. Включите "Удалённый вход".

---

## 4) Получите IP-адрес ВМ

```bash
lume get openclaw
```

Найдите IP-адрес (обычно `192.168.64.x`).

---

## 5) Подключитесь к ВМ по SSH

```bash
ssh youruser@192.168.64.X
```

Замените `youruser` на созданную вами учётную запись, а IP — на IP-адрес вашей ВМ.

---

## 6) Установите OpenClaw

Внутри ВМ:

```bash
npm install -g openclaw@latest
openclaw onboard --install-daemon
```

Следуйте подсказкам при настройке, чтобы указать поставщика модели (Anthropic, OpenAI и т. д.).

---

## 7) Настройте каналы

Отредактируйте файл конфигурации:

```bash
nano ~/.openclaw/openclaw.json
```

Добавьте свои каналы:

```json5
{
  channels: {
    whatsapp: {
      dmPolicy: "allowlist",
      allowFrom: ["+15551234567"],
    },
    telegram: {
      botToken: "YOUR_BOT_TOKEN",
    },
  },
}
```

Затем войдите в WhatsApp (отсканируйте QR-код):

```bash
openclaw channels login
```

---

## 8) Запустите ВМ без графического интерфейса

Остановите ВМ и перезапустите без дисплея:

```bash
lume stop openclaw
lume run openclaw --no-display
```

ВМ работает в фоновом режиме. Демон OpenClaw поддерживает работу шлюза.

Чтобы проверить статус:

```bash
ssh youruser@192.168.64.X "openclaw status"
```

## Бонус: интеграция с iMessage

Это ключевая особенность работы на macOS. Используйте [BlueBubbles](https://bluebubbles.app), чтобы добавить iMessage в OpenClaw.

Внутри ВМ:
1. Загрузите BlueBubbles с сайта bluebubbles.app.
2. Войдите с помощью своего Apple ID.
3. Включите Web API и задайте пароль.
4. Настройте веб-хуки BlueBubbles на ваш шлюз (пример: `https://your-gateway-host:3000/bluebubbles-webhook?password=<password>`).

Добавьте в конфигурацию OpenClaw:

```json5
{
  channels: {
    bluebubbles: {
      serverUrl: "http://localhost:1234",
      password: "your-api-password",
      webhookPath: "/bluebubbles-webhook",
    },
  },
}
```

Перезапустите шлюз. Теперь ваш агент может отправлять и получать iMessage.

Подробные инструкции по настройке: [Канал BlueBubbles](/channels/bluebubbles)

## Сохраните эталонный образ

Перед дальнейшей настройкой создайте снимок чистого состояния:

```bash
lume stop openclaw
lume clone openclaw openclaw-golden
```

Сброс в любое время:

```bash
lume stop openclaw && lume delete openclaw
lume clone openclaw-golden openclaw
lume run openclaw --no-display
```

## Работа 24/7

Чтобы ВМ работала постоянно:
- держите Mac подключённым к сети;
- отключите переход в режим сна в "Системных настройках" → "Экономия энергии";
- при необходимости используйте `caffeinate`.

Для действительно постоянной работы рассмотрите возможность использования выделенного Mac mini или небольшого VPS. См. [хостинг VPS](/vps).

---

## Устранение неполадок

| Проблема | Решение |
| -------- | ------- |
| Не удаётся подключиться к ВМ по SSH | Проверьте, включён ли "Удалённый вход" в "Системных настройках" ВМ |
| IP-адрес ВМ не отображается | Подождите, пока ВМ полностью загрузится, и снова выполните `lume get openclaw` |
| Команда Lume не найдена | Добавьте `~/.local/bin` в ваш PATH |
| QR-код WhatsApp не сканируется | Убедитесь, что вы вошли в ВМ (а не на хост) при выполнении `openclaw channels login` |
