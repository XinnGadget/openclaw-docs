# 実験的機能

OpenClaw の実験的機能は、**オプトインのプレビュー対象**です。これらは、安定したデフォルトや長期的に維持される公開契約に値する前に、実運用での十分な実績がまだ必要なため、明示的なフラグの背後にあります。

通常の設定とは別物として扱ってください。

- 関連ドキュメントで試すよう案内されていない限り、デフォルトでは**オフのまま**にしてください。
- 安定した設定よりも、**形や動作が変わる**ペースが速いことを想定してください。
- すでに安定した手段がある場合は、まずそちらを優先してください。
- OpenClaw を広く展開する場合は、実験的フラグを共有ベースラインに組み込む前に、より小さな環境でテストしてください。

## 現在ドキュメント化されているフラグ

| 対象                     | キー                                                      | 使う場面                                                                                                       | 詳細                                                                                          |
| ------------------------ | --------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| ローカルモデルランタイム | `agents.defaults.experimental.localModelLean`             | より小規模または厳格なローカルバックエンドが、OpenClaw の完全なデフォルトツール対象を処理しきれない場合      | [Local Models](/ja-JP/gateway/local-models)                                                         |
| メモリ検索               | `agents.defaults.memorySearch.experimental.sessionMemory` | `memory_search` が過去のセッション文字起こしをインデックスできるようにし、追加の保存コストや索引コストを許容したい場合 | [メモリ設定リファレンス](/ja-JP/reference/memory-config#session-memory-search-experimental)         |
| 構造化プランニングツール | `tools.experimental.planTool`                             | 互換性のあるランタイムと UI で、複数ステップ作業の追跡用に構造化された `update_plan` ツールを公開したい場合  | [Gateway 設定リファレンス](/ja-JP/gateway/configuration-reference#toolsexperimental)                |

## ローカルモデル lean モード

`agents.defaults.experimental.localModelLean: true` は、弱いローカルモデル構成向けの逃がし弁です。`browser`、`cron`、`message` のような重量級のデフォルトツールを削り、プロンプトの形をより小さくし、小さなコンテキストやより厳格な OpenAI 互換バックエンドでも壊れにくくします。

これは意図的に**通常の経路ではありません**。バックエンドが完全なランタイムを問題なく処理できるなら、これをオフのままにしてください。

## 実験的は隠し機能を意味しない

機能が実験的であるなら、OpenClaw はそれをドキュメント内でも設定パス自体でも、明確にそう示すべきです。逆に、やってはいけないのは、プレビュー動作を安定して見えるデフォルトのつまみに紛れ込ませて、それが普通であるかのように見せることです。そうすると設定対象はすぐに煩雑になります。
