# Runway

В OpenClaw включён встроенный провайдер `runway` для удалённой генерации видео.

- ID провайдера: `runway`
- Аутентификация: `RUNWAYML_API_SECRET` (канонический вариант) или `RUNWAY_API_KEY`
- API: генерация видео на основе задач в Runway (`GET /v1/tasks/{id}` с опросом)

## Быстрый старт

1. Установите API-ключ:

```bash
openclaw onboard --auth-choice runway-api-key
```

2. Сделайте Runway провайдером видео по умолчанию:

```bash
openclaw config set agents.defaults.videoGenerationModel.primary "runway/gen4.5"
```

3. Попросите агента сгенерировать видео. Runway будет использован автоматически.

## Поддерживаемые режимы

| Режим | Модель | Входной референс |
| -------------- | ------------------ | ----------------------- |
| Текст-в-видео | `gen4.5` (по умолчанию) | Нет |
| Изображение-в-видео | `gen4.5` | 1 локальное или удалённое изображение |
| Видео-в-видео | `gen4_aleph` | 1 локальное или удалённое видео |

- Локальные референсы изображений и видео поддерживаются через data URI.
- Для режима "видео-в-видео" в настоящее время требуется именно `runway/gen4_aleph`.
- При генерации только по тексту доступны соотношения сторон `16:9` и `9:16`.

## Конфигурация

```json5
{
  agents: {
    defaults: {
      videoGenerationModel: {
        primary: "runway/gen4.5",
      },
    },
  },
}
```
