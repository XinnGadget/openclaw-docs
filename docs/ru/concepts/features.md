## Полный список

**Каналы:**

- Встроенные каналы: Discord, Google Chat, iMessage (устаревший), IRC, Signal, Slack, Telegram, WebChat, WhatsApp;
- Каналы со встроенными плагинами: BlueBubbles для iMessage, Feishu, LINE, Matrix, Mattermost, Microsoft Teams, Nextcloud Talk, Nostr, QQ Bot, Synology Chat, Tlon, Twitch, Zalo, Zalo Personal;
- Дополнительные плагины каналов, устанавливаемые отдельно: Voice Call и сторонние пакеты, например WeChat;
- Сторонние плагины каналов могут дополнительно расширить возможности шлюза (например, WeChat);
- Поддержка групповых чатов с активацией по упоминаниям;
- Безопасность личных сообщений с использованием белых списков и сопряжения.

**Агент:**

- Встроенная среда выполнения агента с потоковой передачей инструментов;
- Маршрутизация между агентами с изолированными сессиями для каждого рабочего пространства или отправителя;
- Сессии: прямые чаты объединяются в общий `main`, групповые чаты изолированы;
- Потоковая передача и разбиение на фрагменты для длинных ответов.

**Аутентификация и провайдеры:**

- Более 35 провайдеров моделей (Anthropic, OpenAI, Google и другие);
- Аутентификация по подписке через OAuth (например, OpenAI Codex);
- Поддержка пользовательских и самостоятельно размещённых провайдеров (vLLM, SGLang, Ollama и любых конечных точек, совместимых с OpenAI или Anthropic).

**Медиа:**

- Приём и отправка изображений, аудио, видео и документов;
- Общие возможности генерации изображений и видео;
- Транскрипция голосовых заметок;
- Преобразование текста в речь с использованием нескольких провайдеров.

**Приложения и интерфейсы:**

- WebChat и веб-интерфейс управления в браузере;
- Сопутствующее приложение для панели меню macOS;
- Узел для iOS с возможностью сопряжения, Canvas, камерой, записью экрана, определением местоположения и голосовым общением;
- Узел для Android с возможностью сопряжения, чатом, голосовым общением, Canvas, камерой и командами для устройства.

**Инструменты и автоматизация:**

- Автоматизация работы с браузером, выполнение команд, песочница;
- Веб-поиск (Brave, DuckDuckGo, Exa, Firecrawl, Gemini, Grok, Kimi, MiniMax Search, Ollama Web Search, Perplexity, SearXNG, Tavily);
- Задания Cron и планирование по таймеру;
- Навыки, плагины и конвейеры рабочих процессов (Lobster).
