# GitHub Copilot

## Что такое GitHub Copilot?

GitHub Copilot — это помощник по программированию на основе ИИ от GitHub. Он предоставляет доступ к моделям Copilot для вашей учётной записи GitHub и тарифного плана. OpenClaw может использовать Copilot в качестве поставщика моделей двумя способами.

## Два способа использования Copilot в OpenClaw

### 1) Встроенный поставщик GitHub Copilot (`github-copilot`)

Используйте встроенный поток входа через устройство, чтобы получить токен GitHub, а затем обменяйте его на токены API Copilot при запуске OpenClaw. Это **стандартный** и самый простой способ, поскольку он не требует VS Code.

### 2) Плагин Copilot Proxy (`copilot-proxy`)

Используйте расширение **Copilot Proxy** для VS Code в качестве локального моста. OpenClaw взаимодействует с конечной точкой `/v1` прокси и использует список моделей, который вы там настроите. Выберите этот способ, если вы уже используете Copilot Proxy в VS Code или вам нужно маршрутизировать запросы через него. Необходимо активировать плагин и поддерживать работу расширения VS Code.

Используйте GitHub Copilot в качестве поставщика моделей (`github-copilot`). Команда входа запускает поток устройств GitHub, сохраняет профиль аутентификации и обновляет вашу конфигурацию для использования этого профиля.

## Настройка через CLI

```bash
openclaw models auth login-github-copilot
```

Вам будет предложено перейти по URL-адресу и ввести одноразовый код. Не закрывайте терминал до завершения процесса.

### Опциональные флаги

```bash
openclaw models auth login-github-copilot --yes
```

Чтобы одновременно применить рекомендуемую поставщиком модель по умолчанию, используйте общую команду аутентификации:

```bash
openclaw models auth login --provider github-copilot --method device --set-default
```

## Установка модели по умолчанию

```bash
openclaw models set github-copilot/gpt-4o
```

### Фрагмент конфигурации

```json5
{
  agents: { defaults: { model: { primary: "github-copilot/gpt-4o" } } },
}
```

## Примечания

- Требуется интерактивный TTY; запускайте команду непосредственно в терминале.
- Доступность моделей Copilot зависит от вашего тарифного плана; если модель отклонена, попробуйте другой идентификатор (например, `github-copilot/gpt-4.1`).
- Идентификаторы моделей Claude автоматически используют транспорт Anthropic Messages; модели GPT, серии o и Gemini используют транспорт OpenAI Responses.
- При входе токен GitHub сохраняется в хранилище профилей аутентификации и обменивается на токен API Copilot при запуске OpenClaw.
