# Выполнение кода

`code_execution` запускает удалённый анализ на Python в песочнице через API ответов xAI.
Это отличается от локального [`exec`](/tools/exec):

- `exec` выполняет команды оболочки на вашем компьютере или узле;
- `code_execution` выполняет Python в удалённой песочнице xAI.

Используйте `code_execution` для:

- вычислений;
- составления таблиц;
- быстрой статистики;
- анализа в виде диаграмм;
- анализа данных, возвращённых `x_search` или `web_search`.

**Не** используйте его, если вам нужны локальные файлы, ваша оболочка, ваш репозиторий или сопряжённые устройства. Для этого используйте [`exec`](/tools/exec).

## Настройка

Вам нужен API-ключ xAI. Подойдёт любой из следующих:

- `XAI_API_KEY`;
- `plugins.entries.xai.config.webSearch.apiKey`.

Пример:

```json5
{
  plugins: {
    entries: {
      xai: {
        config: {
          webSearch: {
            apiKey: "xai-...",
          },
          codeExecution: {
            enabled: true,
            model: "grok-4-1-fast",
            maxTurns: 2,
            timeoutSeconds: 30,
          },
        },
      },
    },
  },
}
```

## Как использовать

Формулируйте запрос естественным образом и чётко обозначайте намерение провести анализ:

```text
Используйте code_execution, чтобы рассчитать скользящее среднее за 7 дней для следующих чисел: ...
```

```text
Используйте x_search, чтобы найти посты с упоминанием OpenClaw за эту неделю, затем используйте code_execution, чтобы подсчитать их по дням.
```

```text
Используйте web_search, чтобы собрать последние данные о бенчмарках ИИ, затем используйте code_execution, чтобы сравнить процентные изменения.
```

Инструмент внутренне принимает единственный параметр `task`, поэтому агент должен отправлять полный запрос на анализ и любые встроенные данные в одном промпте.

## Ограничения

- Это удалённое выполнение в xAI, а не выполнение локального процесса.
- Его следует рассматривать как временный анализ, а не как постоянный блокнот.
- Не предполагайте наличие доступа к локальным файлам или вашему рабочему пространству.
- Чтобы получить актуальные данные X, сначала используйте [`x_search`](/tools/web#x_search).
