# WebChat (приложение для macOS)

Приложение в строке меню macOS встраивает пользовательский интерфейс WebChat как нативное представление SwiftUI. Оно подключается к шлюзу (Gateway) и по умолчанию использует **основную сессию** для выбранного агента (с переключателем сессий для других сессий).

- **Локальный режим**: подключается напрямую к локальному WebSocket шлюза (Gateway).
- **Удаленный режим**: перенаправляет порт управления шлюза (Gateway) через SSH и использует этот туннель в качестве плоскости данных.

## Запуск и отладка

- Вручную: меню Lobster → "Open Chat".
- Автоматическое открытие для тестирования:

  ```bash
  dist/OpenClaw.app/Contents/MacOS/OpenClaw --webchat
  ```

- Логи: `./scripts/clawlog.sh` (подсистема `ai.openclaw`, категория `WebChatSwiftUI`).

## Как это устроено

- Плоскость данных: методы Gateway WS `chat.history`, `chat.send`, `chat.abort`, `chat.inject` и события `chat`, `agent`, `presence`, `tick`, `health`.
- `chat.history` возвращает строки расшифровки, нормализованные для отображения: теги встроенных директив удаляются из видимого текста, полезные нагрузки XML вызовов инструментов в виде обычного текста (включая `<tool_call>...</tool_call>`, `<function_call>...</function_call>`, `<tool_calls>...</tool_calls>`, `<function_calls>...</function_calls>` и усечённые блоки вызовов инструментов) удаляются, просочившиеся токены управления моделью в формате ASCII/полной ширины удаляются, строки помощника, состоящие исключительно из тихих токенов (например, точные `NO_REPLY` / `no_reply`), опускаются, а слишком длинные строки могут быть заменены на плейсхолдеры.
- Сессия: по умолчанию используется основная сессия (`main` или `global`, если область действия глобальная). Пользовательский интерфейс позволяет переключаться между сессиями.
- Для первичной настройки используется выделенная сессия, чтобы отделить процесс первой настройки от остального функционала.

## Поверхность безопасности

- В удалённом режиме через SSH перенаправляется только порт управления WebSocket шлюза (Gateway).

## Известные ограничения

- Пользовательский интерфейс оптимизирован для чат‑сессий (не является полноценной песочницей браузера).
