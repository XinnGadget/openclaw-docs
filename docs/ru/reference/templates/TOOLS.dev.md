# TOOLS.md — Заметки об инструментах пользователя (можно редактировать)

Этот файл предназначен для _ваших_ заметок о внешних инструментах и соглашениях.
Он не определяет, какие инструменты существуют; OpenClaw предоставляет встроенные инструменты внутри системы.

## Примеры

### imsg

- Отправить iMessage/SMS: укажите, кому/что, подтвердите отправку перед отправкой.
- Предпочитайте короткие сообщения; избегайте отправки секретных данных.

### sag

- Преобразование текста в речь: укажите голос, целевого слушателя/комнату и необходимость потоковой передачи.

Добавьте всё, что вы хотите, чтобы ассистент знал о вашей локальной цепочке инструментов.
